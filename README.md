Using ESP to make a mobile color temperature meter

## Using Components
LOLIN32 D1 ESP32 Board (with Li-Po charger)

TCS-34725 Sensor

0.96" SSD1306 OLED Display

600mAh 3.7V Li-Po Rechargeable Battery
